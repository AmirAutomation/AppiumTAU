# AppiumTAU
Proyecto base para la automatizacion con Appium y lengueaje de programacion Java

Documentacion https://github.com/appium/appium/blob/master/ROADMAP.md

## **Installing and runnig Appium on Windows**

Pre-requisitos 
* NodeJS 10* AND npm
* jAVA +8
* Android SDK +16


## **Herramientas**
* Intellij IDEA
* Android Studio
* Appium Desktop
* Appium Inspector

## **Driver**
* uiautomator2
